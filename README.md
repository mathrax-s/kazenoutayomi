# 風のうたよみ


## 作品の紹介

[![風のうたよみ](http://img.youtube.com/vi/Kt1RpMQv_q4/0.jpg)](http://www.youtube.com/watch?v=Kt1RpMQv_q4 "風のうたよみ")

六甲オルゴールミュージアムの中庭に、鳥の木彫が10体おいてあります。

その鳥の背中を撫でると、オルゴールの音色の音や、風の流れるような響が聞こえます。
